# Ma carte de visite virtuelle
Petit projet perso : une v-card 100% HTML/CSS, sans framework, juste pour le plaisir et pour pratiquer

### Objectif du projet 
Me frabiquer une V-Card en ligne propre 
- HTML sémantique & accessibilité
- Mise en page avancée avec Flexbox
- Bonne pratique Git : Travail et push sur la branche

### Fonctionnalités
- 100% statique (HTML + CSS uniquement)
- Déploiement automaqiue avec GitHub Pages

## Technologies
- HTML
- CSS
- Git
